# 🔍 Social Graph Recommendation System

## 📌 Overview

Python-based **social network analysis & recommendation engine** built using JSON datasets.
Implements core platform features like:

* 👥 People You May Know (mutual friends)
* 📄 Pages You Might Like (shared interests)
* 🧹 Data cleaning pipeline

---

## 🚀 Features

* **Data Cleaning:** removes invalid users, duplicates, inactive profiles
* **User Recommendations:** mutual-friend based ranking
* **Page Recommendations:** interest similarity scoring
* Efficient use of **sets, dictionaries, and sorting**

---

## 🧠 Tech & Concepts

`Python` • `JSON` • `Set Operations` • `Graph Logic` • `Ranking Algorithms`
